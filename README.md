# 김민재 이력서 / 포트폴리오

이 레포는 GitHub Pages로 공개하는 이력서와 포트폴리오 문서를 담고 있습니다.

## 바로 보기

- 포트폴리오: [minzai0116.github.io/resume-portfolio/portfolio/](https://minzai0116.github.io/resume-portfolio/portfolio/)
- 이력서: [minzai0116.github.io/resume-portfolio/resume/](https://minzai0116.github.io/resume-portfolio/resume/)

## 구성

- `portfolio/`: 대표 프로젝트의 문제 정의, 역할, 구현 범위를 정리한 포트폴리오
- `resume/`: 핵심 역량, 기술 스택, 대표 프로젝트를 압축한 이력서

## 소개

- 추천 시스템, 검색/리랭킹, LLM 챗봇, 데이터 기반 서비스 구현 경험을 중심으로 구성했습니다.
- `fcoach`, `RMSE`, `Sound_of_Heart`를 대표 프로젝트로 먼저 보여주도록 정리했습니다.
